安装

docsite 用于根据markdown文档快速搭建静态站点。

  1. 确保安装的node的版本满足如下要求：node >= 6.9.0
  2. 执行npm install docsite -g执行全局安装

验证安装

在终端中执行docsite -V，能够显示当前安装的docsite版本，则表示安装成功。
